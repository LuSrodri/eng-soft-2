
## Description

A simple api made with [Nest](https://github.com/nestjs/nest).

## Installation

```bash
$ npm install
```

## Running the app
In a machine with docker and docker compose, follow the below steps.
 - Set the environment variables:
```
PORT="3000"
ENVIRON="DEV"
```
 - Run:
```bash
$ docker compose up
```
 - Access localhost:3000 and enjoy it :)

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
